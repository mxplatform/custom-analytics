# Custom Analytics

This repository contains custom analytics tools and scripts for internal data analysis and benchmarking.

## Folder Structure

- `dbt_benchmark/`  
  Contains benchmarking scripts and resources for dbt projects.
- schema/
  Contains the benchmark table schema for review

## Getting Started

1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/your-repo-name.git](https://github.com/mxplatform/Custom-Analytics.git)
   ```

2. Install dependencies (if any):
   ```bash
   # Example for Python projects
   pip install -r requirements.txt
   ```

3. Run dbt modules.
